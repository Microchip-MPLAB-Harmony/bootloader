bootloader/tools/btl_bin_to_hex_gen.py 

  • This script should be used to convert the binary output to a C style array containing Hex output. It is mainly when programming the application using the bootloader_host_nvm host application
  • If size of the input binary file is not aligned to device erase boundary it appends 0xFF to the binary to make it aligned and then generates the Hex output
  • User must specify the binary file to convert (-b), hex output file (-o) and the device (-d)

 

  • Compatible with Python 3.x

 

bootloader/tools/btl_app_merge_bin.py  

  • This script should be used to merge the bootloader binary and application binary
  • It creates a merged binary output where bootloader is placed from start and the application will be placed at the offset passed as parameter
  • If the application offset is not equal to end of bootloader offset it fills the gap with 0xFF until the application offset
  • The merged binary will be used by btl_host.py as input for fail safe update

 

  • Compatible with Python 3.x

 

 

Example usage: 

 

btl_bin_to_hex_gen.py 

 

python <harmony3_path>\bootloader\tools\btl_bin_to_hex_gen.py --help

 

 

 

python <harmony3_path>\bootloader\tools\btl_bin_to_hex_gen.py -b <binary_file> -o <hex_file> -d <device> 

 

python <harmony3_path>\bootloader\tools\btl_bin_to_hex_gen.py -b D:\harmony\bootloader\apps\i2c_bootloader\test_app\firmware\sam_d20_xpro.X\dist\sam_d20_xpro\production\sam_d20_xpro.X.production.bin -o D:\harmony\bootloader\apps\i2c_bootloader\bootloader_host_nvm\firmware\src\test_app_images\image_pattern_hex_sam_d20_xpro.h -d samd2x

 

 

Fail Safe Update Mode Usage Example

 

btl_app_merge_bin.py 

python <harmony3_path>\bootloader\tools\btl_app_merge_bin.py --help

 

 

 

python <harmony3_path>\bootloader\tools\btl_app_merge_bin.py -o 0x2000 -b <harmony3_path>\bootloader\apps\i2c_bootloader\fail_safe_bootloader\bootloader\firmware\sam_e54_xpro.X\dist\sam_e54_xpro\production\sam_e54_xpro.X.production.bin -a <harmony3_path>\bootloader\apps\i2c_bootloader\fail_safe_bootloader\test_app\firmware\sam_e54_xpro.X\dist\sam_e54_xpro\production\sam_e54_xpro.X.production.bin
