bootloader

This application is a fail safe bootloader application which resides in the starting location of the both the banks of device flash memory

  • The application calls the bootloader_Task() which receives application to be programmed into opposite bank of flash memory over I2C interface
  • This application uses the On board Switch as bootloader trigger pin to force enter the bootloader at reset of device.

 

Following embedded host application example is provided at the below given paths to program the user application using the bootloader.
