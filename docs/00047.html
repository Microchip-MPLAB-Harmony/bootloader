bootloader_host_nvm

The user application binary must be converted to a header file containing the application image in HEX format in a C style array. A btl_bin_to_hex_gen.py utility is provided to do this conversion.

  • Add the generated application image header file is to the bootloader host application project. Rebuild and program the host bootloader application. This results in the application image being copied in the host MCU's flash (NVM).
  • Press the user switch on the target board to trigger bootloader.
  • Press the user switch on the host board to start the programming operation. The host bootloader application sends the application image stored in it's internal flash (NVM) to the target board over the I2C bus and then swaps the flash banks.
