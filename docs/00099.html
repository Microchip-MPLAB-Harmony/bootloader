PIC32 Device
  1. Connect a micro USB cable to the DEBUG port.
  2. Build and program the uart fail safe bootloader application using the MPLAB X IDE.
  3. Build the Application test_app using MPLAB X IDE. Do not program
  4. Refer to Bootloader Tools Help for setting up the host script
  5. Run the btl_host.py from command prompt to program the application binary to opposite panel.

 

python <harmony3_path>\bootloader\tools\btl_host.py -v -s -i <COM PORT> -d <Device Name> -a <Address> -f <harmony3_path>\bootloader\apps\uart_fail_safe_bootloader\test_app\firmware\<Project Name>\dist\<Config Name>\production\<Project Name>.production.bin
